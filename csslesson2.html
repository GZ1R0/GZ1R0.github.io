What is a div element?

The div element is like a bucket that groups sections of a webpage together.


What is one way you can identify or "target" a div element?

A div can be identified or targeted in several different ways. We're focusing on classes for now. When you are calling a class in CSS, it will target every div with the same class. You can repeat styling across a website using the same class without having to repeat the CSS.


Why would you use a div element?

To group content so it can be easily styled using the class or id attributes, marking a section of a document as being written in a different language (using the lang attribute), and so on.


What is a border property?

The border property allows you to style the border around an element.


What are some of the different border properties?

  • Border width
  • Border color
  • Border sides
  • Rounded borders


What is the difference between margins and padding?

A margin is the spacing around an element from everything around it. The margin begins at the element's border.
Padding is the spacing content has from the element border.
Think of padding as someone wearing a big coat, and the margin as the space people give them.
